Embedding using iframes

Embed in a snap

You can embed a single Qlik Sense object on a web page using the Single configurator.Embedding a Qlik Sense visualization is as easy as inserting a line of code. Here's the structure of the code snippet:

<iframe src='http://<your QSE host>/single/?appid=<yourappid>&obj<yourobject>&opt=<your selected options>' style='border:none;width:<your width>;height:<your height>;'></iframe>
Embedding Options

You can optionally embed elements like:

  • Interactions
  • Selections
  • Animations
  • Context menu
  • Selection Bar
  • Set clear selections
